## Image Guidelines

- **Format**: JPG, PNG, or WebP
- **Size**: Recommended 800x1000px (portrait orientation)
- **File Size**: Keep under 500KB each (compress if needed)
- **Aspect Ratio**: 4:5 works best for the slider

## Compression Tools

- [TinyPNG](https://tinypng.com) - Online compression
- [ImageOptim](https://imageoptim.com) - Mac app
- [Squoosh](https://squoosh.app) - Google's web app
